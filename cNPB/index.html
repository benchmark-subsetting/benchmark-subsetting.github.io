NAS-C-OpenMP3.0

We maintain an unofficial C version of the NPB 3.0 OpenMP that is built upon the NPB OMP 2.3 C version from the Omni Compiler Project. Informations on the original NAS Parallel Benchmarks are available at NASA NPB project page.

The benchmarks were modified to match the new official 3.0 Fortran NPB benchmarks. In particular, benchmarks in this release follow the same parallel region structure than the official 3.0 Fortran NPB.
