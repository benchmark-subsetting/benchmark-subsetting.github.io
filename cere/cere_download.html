Installing CERE

Prerequisites

CERE requires LLVM and Clang compiler. It works best with LLVM 3.3

  • You can download LLVM and all its components here: LLVM 3.3
  • Informations on how to extracte and build LLVM can be found here: Building and installing LLVM
  • You also need the DragonEgg plugin to compile Fortran code with LLVM: DragonEgg
  • If you have difficulties compiling DragonEgg with a GCC > 4.7, consider using GCC 4.7: "update-alternatives --config gcc" to choose between installed version of GCC (Debian/Ubuntu).
  • Finnally install autotools: "sudo apt-get install autotools-dev automake autoconf libtool" (Debian/Ubuntu).

Installation

  1. Unzip CERE.
  2. ./autogen.sh
  3. make
  4. To test if everything is working:
    1. cd tests
    2. ./test.sh

Known Issues

Problems compiling Fortran code with CERE:

  • Check that DRAGONEGG_PATH in CERE/src/lec/lec is correctly pointing to the file dragonegg.so
